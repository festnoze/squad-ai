<contexte>
Il s'agit d'identifier, d'ordonnancer, puis de renvoyer sous forme d'un tableau json toutes les valeurs pour chaque champ du bloc de question indiqué.
Un bloc de question est composé de un ou plusieurs champs pour lesquelles une valeur doit être fournie par l'utilisateur. 
</contexte>

<contexte>
Le bloc de questions concerné se nomme : '{group_name}'
Ce bloc de questions a pour description : '{group_desc}'.
Voici la liste des champs contenus dans ce bloc de questions, pour laquelle l'utilisateur doit fournir une réponse (chaque question est à traduire en français si besoin):
{fields_infos}
</contexte>

<objectifs>
Objectifs:
1- Extraire de la réponse de l'utilisateur toutes les valeurs fournies,
2- Trouver la correspondance de chaque valeur avec le champ qui lui correspond, si aucun champ correspondant n'est trouvé, affecte la valeur 'null'.
3- Réordonner les réponses pour correspondre à l'ordre des champs du bloc de questions,
4- Renvoier les réponses sous forme d'un tableau de string contenant les valeurs réordonnées fournies par l'utilisateur.
</objectifs>
