## ML-Specific Metrics to Include in Bullets

| Category | Metrics |
|----------|---------|
| Model performance | Accuracy, Precision, Recall, F1-score, AUC-ROC, BLEU, ROUGE, Perplexity |
| Latency | p50, p95, p99 inference time |
| Scale | Daily predictions served, dataset size, training time |
| Business impact | Revenue generated, cost saved, manual hours eliminated |
| Improvement | "from X to Y" format: "Improved F1 from 0.78 to 0.93" |

## Project Showcase Format

```
[Project Name] | [Tech Stack]                                    [Date]
- One-sentence description of what was built and why
- "Trained [model type] on [dataset size] achieving [metric]: [value]"
- "Deployed to [platform] serving [N] predictions/day at [latency] p95"
- "Impact: [business outcome with number]"
- Link: github.com/user/repo | huggingface.co/user/model
```
