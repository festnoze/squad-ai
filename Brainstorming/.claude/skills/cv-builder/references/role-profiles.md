# Role-Specific CV Profiles

## Full Stack Developer / Engineer

### What Recruiters Look For
- End-to-end ownership: frontend + backend + database + deployment
- Ability to ship features independently, not just "contribute to"
- Modern stack fluency: React/Next.js/Vue + Node/Python/Go + SQL/NoSQL + Cloud
- System design thinking: API design, caching, auth, CI/CD

### Key Technical Skills to Highlight

```
Frontend:         React, Next.js, Vue.js, TypeScript, Tailwind CSS, HTML5/CSS3
Backend:          Node.js, Python (FastAPI/Django), Go, REST API, GraphQL, gRPC
Databases:        PostgreSQL, MongoDB, Redis, Elasticsearch
Cloud & DevOps:   AWS/GCP/Azure, Docker, Kubernetes, Terraform, CI/CD (GitHub Actions)
Testing:          Jest, Pytest, Cypress, Playwright, E2E testing
Auth & Security:  OAuth2, JWT, RBAC, OWASP Top 10
```

### Bullet Examples (XYZ Format)

```
Architected and deployed a full-stack SaaS platform (React + FastAPI + PostgreSQL)
serving 15K monthly active users with 99.9% uptime on AWS ECS.

Reduced page load time by 68% (from 4.2s to 1.3s) by implementing server-side rendering
with Next.js and optimizing database queries with proper indexing.

Built real-time collaborative editing feature using WebSockets and CRDT,
enabling 50+ concurrent users per document with sub-200ms sync latency.

Designed and implemented RESTful API serving 2M+ requests/day with comprehensive
OpenAPI documentation, reducing frontend-backend integration time by 40%.

Led migration from monolithic Django app to microservices architecture (12 services),
reducing deployment time from 45min to 8min and enabling independent team releases.
```

### Summary Example

> Full Stack Engineer with 6+ years building production web applications end-to-end.
> Architected a SaaS platform serving 15K MAU on AWS, achieving 99.9% uptime.
> Expert in React/TypeScript frontends backed by Python/Node APIs and PostgreSQL.
> Passionate about developer experience, performance optimization, and shipping fast.

- MLOps maturity: experiment tracking, model versioning, A/B testing

## Forward Deployed Engineer (FDE)

### What the Role Is
FDEs work directly with customers to deploy, customize, and integrate a company's product.
They combine software engineering with customer-facing skills, technical consulting,
and rapid prototyping. Common at companies like Palantir, Scale AI, Databricks, Weights & Biases.

### What Recruiters Look For
- **Technical breadth** over depth: comfortable across the full stack + data + infra
- **Customer-facing skills**: presenting to executives, translating business needs to technical specs
- **Speed of execution**: building MVPs and POCs in days, not months
- **Adaptability**: switching between industries (healthcare, defense, finance, logistics)
- **Problem-solving in ambiguity**: working without clear specs, in messy data environments
- **Integration skills**: connecting APIs, data pipelines, and legacy systems

### Key Technical Skills to Highlight

```
Languages:        Python, TypeScript, SQL, Java (for enterprise clients)
Full Stack:       React/Next.js, Node.js, FastAPI, PostgreSQL
Data & Analytics: Spark, Airflow, Pandas, SQL, ETL pipelines, data modeling
Cloud:            AWS, GCP, Azure (multi-cloud is a plus)
Integration:      REST APIs, GraphQL, Webhooks, SSO/SAML, legacy system connectors
Tools:            Docker, Git, Jupyter, Terraform, CI/CD
Soft Skills:      Client communication, technical presentations, rapid prototyping,
                  stakeholder management, requirements gathering
```

### Bullet Examples (XYZ Format)

```
Deployed product to 3 Fortune 500 clients in healthcare and finance, leading
end-to-end technical integration and delivering $4.2M in first-year contract value.

Built custom data pipeline connecting client's legacy Oracle DB to platform's API,
processing 2.3M records/day and reducing manual data entry by 95%.

Led 12 client workshops translating business requirements into technical specifications,

integration toolkit with automated data validation and error reporting.
```

### How FDE Differs from Other Roles

| Aspect | SWE | AI Engineer | FDE |
|--------|-----|-------------|-----|
| Primary output | Product features | ML models/pipelines | Client deployments |
| Audience | Internal team | Internal/research | External clients |
| Key metric | Code quality, velocity | Model performance | Client success, revenue |
| Breadth vs depth | Deep in 1-2 areas | Deep in ML | Broad across stack |
| Soft skills weight | Medium | Medium | Very high |
| Ambiguity tolerance | Low-medium | Medium | Very high |
