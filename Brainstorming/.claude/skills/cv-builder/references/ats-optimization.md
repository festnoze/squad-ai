# ATS Optimization Guide

Over 90% of employers use Applicant Tracking Systems. 99.7% of recruiters use keyword filters.

## Layout Rules (Mandatory)

| Rule | Details |
|------|---------|
| Columns | Single column only. Multi-column layouts scramble in ATS. |
| Tables | Never. ATS cannot parse table content reliably. |
| Text boxes | Never. Content inside them is often invisible to ATS. |
| Headers/Footers | All contact info in main body, never in document headers/footers. |
| Graphics/Images | No logos, icons, charts, skill bars, progress bars, or photos. |

## Section Headers (Use Exactly These)

- PROFESSIONAL SUMMARY (or SUMMARY)
- WORK EXPERIENCE (or PROFESSIONAL EXPERIENCE)
- SKILLS (or TECHNICAL SKILLS)
- EDUCATION
- CERTIFICATIONS
- PROJECTS
- PUBLICATIONS
- AWARDS

Never use creative headers like "Where I've Made an Impact" or "My Journey."

## Typography

- **Safe fonts**: Arial, Calibri, Aptos, Helvetica, Georgia, Times New Roman, Verdana
- **Body text**: 10-12pt
- **Section headers**: 14-16pt, bold
- **Name**: 18-24pt
- **Max 2 font families** (one for headings, one for body); single font with weight variations is safest
- **Bullet symbols**: Standard round bullet or dash (-) only. No checkmarks, stars, arrows.

## Color

- One accent color maximum: navy (#2C3E50), dark teal (#0D7377), or charcoal
- Black text for all body content
- Avoid bright colors -- poor print quality and unprofessional signal

## File Format

- Submit as **.docx** unless posting explicitly asks for PDF
- If PDF required, ensure it is text-based (selectable text, not scanned image)
- Never: .pages, .odt, image PDFs

## Keyword Strategy

- Target **15-25 keywords** from the job description
- Include **exact job title** from posting (10.6x higher interview rate)
- Use **both acronyms and full forms**: "Natural Language Processing (NLP)"
- Place keywords in **multiple sections**: Summary, Skills, Experience, Projects
- Use each keyword **2-3 times** in natural context
- Never keyword-stuff -- modern ATS detects and penalizes it

### Aim for 80%+ ATS match rate.

## Pre-Submission Checklist

1. Single-column layout, no tables/text boxes/graphics
2. Standard section headers
3. Contact info in document body, not header/footer
4. .docx format (or text-based PDF)
5. Safe fonts, 10-12pt body
6. Standard bullet characters only
7. 15-25 keywords matching job description
8. Both acronyms and full forms included
9. No special characters, icons, or emojis
10. Consistent date formatting (Mon YYYY - Mon YYYY)
11. Exact job title from posting appears in resume
12. Each keyword used 2-3 times across sections

## Recommended Testing Tools

- **Jobscan** -- Matches resume against specific job descriptions and ATS systems
- **Resume Worded** -- Overall quality + LinkedIn optimization
- **Teal** -- Combined job tracking + resume optimization
