# Action Verbs for Tech Resumes

Use past tense for previous roles, present tense for current role.
Never repeat the same verb in consecutive bullets.

## Engineering & Development

Architected, Automated, Built, Coded, Configured, Debugged, Deployed, Designed,
Developed, Engineered, Implemented, Integrated, Migrated, Modernized, Programmed,
Prototyped, Rebuilt, Re-architected, Refactored, Restructured, Scaffolded, Tested, Validated

## Optimization & Problem-Solving

Accelerated, Decreased, Diagnosed, Eliminated, Enhanced, Improved, Minimized,
Optimized, Overhauled, Reduced, Refined, Resolved, Revamped, Simplified,
Streamlined, Troubleshot, Tuned

## Data & Analysis

Analyzed, Classified, Clustered, Collected, Correlated, Evaluated, Extracted,
Forecasted, Interpreted, Measured, Modeled, Predicted, Processed, Quantified,
Segmented, Synthesized, Transformed, Visualized

## Leadership & Collaboration

Advised, Championed, Coached, Collaborated, Coordinated, Directed, Drove,
Facilitated, Guided, Led, Managed, Mentored, Orchestrated, Partnered, Pioneered,
Spearheaded, Supervised, Trained

## Communication & Documentation

Authored, Briefed, Communicated, Demonstrated, Documented, Presented, Published,
Reported, Wrote

## Innovation & Creation

Conceptualized, Created, Devised, Discovered, Established, Founded, Initiated,
Innovated, Introduced, Invented, Launched, Originated, Pioneered, Proposed

## Scale & Growth

Expanded, Generated, Grew, Increased, Scaled, Strengthened, Surpassed

## Verbs to NEVER Use

- "Responsible for" -> use Managed, Led, Owned
- "Helped with" -> use Collaborated on, Contributed to (with specifics)
- "Worked on" -> use Engineered, Developed, Architected
- "Handled" -> use Orchestrated, Streamlined, Executed
- "Participated in" -> use Drove, Co-led, Spearheaded
- "Did", "Made", "Was involved in" -> always replace
