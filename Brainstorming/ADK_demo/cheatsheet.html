Tools (fonctions Python)

def save_code(code: str) -> dict:
    """Docstring = description pour le LLM.

    Args:
        code: Le code source.
    """
    return {"status": "success", ...}
  • Nom + docstring + type hints = schema LLM
  • Retour prefere : dict avec "status"
  • Non-dict auto-wrappe en {"result": valeur}
  • Pas de ToolContext dans le schema LLM (auto-injecte)

ToolContext & State

from google.adk.tools.tool_context import ToolContext

def my_tool(x: str, tool_context: ToolContext):
    # Lire
    val = tool_context.state.get("key", default)
    # Ecrire
    tool_context.state["key"] = val
    # Escalate (LoopAgent)
    tool_context.actions.escalate = True

Workflow Agents pas de model/instruction

SequentialAgent

SequentialAgent(
  name="pipeline",
  sub_agents=[a, b, c],
)
# a -> b -> c (fixe)

ParallelAgent

ParallelAgent(
  name="parallel",
  sub_agents=[a, b, c],
)
# a + b + c (concurrent)

LoopAgent

LoopAgent(
  name="loop",
  sub_agents=[a, b],
  max_iterations=5,
)
# a->b->a->b... (escalate)

Data flow : output_key + {state_var}

writer = Agent(output_key="code")       # ecrit state["code"]
reviewer = Agent(instruction="Review: {code}")  # lit state["code"]

Callbacks guardrails

from google.adk.agents.context import Context
from google.adk.models import LlmRequest, LlmResponse
from google.adk.tools.base_tool import BaseTool
CallbackSignatureReturn
before_model(Context, LlmRequest)None | LlmResponse
after_model(Context, LlmResponse)None | LlmResponse
before_tool(BaseTool, dict, Context)None | dict
after_tool(BaseTool, dict, Context, dict)None | dict

None = laisser passer  |  Retourner une valeur = bloquer/remplacer
