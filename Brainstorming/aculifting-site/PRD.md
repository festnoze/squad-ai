## 2. Vision Produit

### 2.1 Objectif
Créer un site vitrine **one-page HTML** pour Yamina Heinrich, praticienne en aculifting à [ville], qui :
- Présente ses services d'aculifting de manière professionnelle et rassurante
- Génère des prises de contact et des rendez-vous
- Reflète l'identité visuelle zen/bien-être de sa carte de visite
- Soit facilement modifiable (textes et images) via prompting

### 2.2 Cible
- Femmes 30-65 ans soucieuses de leur apparence
- Cherchant des alternatives naturelles au botox/chirurgie
- Localisation géographique proche du cabinet

### 2.3 Proposition de valeur
> "Redonnez éclat et jeunesse à votre visage naturellement" — technique non invasive, sans injection, sans chirurgie

## 6. Spécifications Techniques

### 6.1 Stack technique
- **HTML5** single file avec CSS et JS intégrés (inline)
- **CSS** : Custom properties (variables), Flexbox, CSS Grid
- **JS** : Vanilla JavaScript (pas de framework)
- **Fonts** : Google Fonts (Playfair Display, Cormorant Garamond, Lato)
- **Icônes** : SVG inline (line-art)

### 6.2 Responsive Design
- **Desktop** : > 1024px — layout 2 colonnes, navigation horizontale
- **Tablette** : 768-1024px — adaptation des grilles
- **Mobile** : < 768px — layout empilé, menu hamburger, CTA flottant

### 6.3 Performance
- Pas de dépendances externes (sauf Google Fonts)
- Images optimisées (WebP avec fallback)
- Lazy loading pour les images hors viewport
- CSS et JS minifiés dans le fichier HTML

### 6.4 Accessibilité
- Contraste WCAG AA minimum
- Alt text sur toutes les images
- Navigation au clavier
- Liens skip-to-content
- Sémantique HTML5 (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)

### 6.5 SEO
- Meta title : "Yamina Heinrich — Aculifting | Spécialiste lifting naturel du visage"
- Meta description dynamique
- Schema.org (LocalBusiness, MedicalBusiness)
- Open Graph tags
